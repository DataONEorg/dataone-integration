DataONE Java Integration Tests
------------------------------

d1_integration represents a testing library that performs integration tests across the DataONE
system, allowing the java client (d1_libclient_java) to be used to call relevant operations on
both Member Node and Coordinating Node instances.

See the test classes under src/test for example usage.
